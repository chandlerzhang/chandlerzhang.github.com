Linux 常用命令

修改网卡MAC地址

举例：把网卡en1的mac地址改为64-27-37-01-1E-0F。
命令：

sudo ifconfig en1 ether 64-27-37-01-1E-0F
sudo ifconfig en1 down
sudo ifconfig en1 up #### 当前IP `ifconfig  | grep 'inet addr:'| grep -v '127.0.0.1' | cut -d: -f2 | awk '{ print $1}'`   或   `ifconfig eth0 | grep inet | awk '{ print $2 }'` #### 创建软链接 `ln -s /Users/mac/Downloads/apache-maven-3.1.0 /usr/share/maven` #### 文件权限

sudo chown root:wheel my.cnf
sudo chmod 644 my.cnf #### 管理进程

ps aux | less
ps aux | grep mysql
sudo kill -9 2319
