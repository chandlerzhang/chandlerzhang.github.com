常用命令

jekyll build  # 构建
jekyll serve  # 启动服务，可在<http://localhost:4000>访问
jekyll build --drafts # 构建草稿

textile: http://textile.sitemonks.com/
liquid语法介绍: http://docs.shopify.com/themes/liquid-basics
变量介绍：http://jekyllrb.com/docs/variables/
遍历BLOG举例如下： 

{% for node in site.posts %}
<div class="row summary">
  <div class="row">
    <div class="col-xs-10">
      <h3>
        <a href="{{node.url}}" class="summary-title">{{node.title}}</a>
      </h3>
    </div>
    <div class="col-xs-2 text-right summary-date">
      <span>{{node.date | date: "%Y-%m-%d"}}</span>
    </div>
  </div>
  <div class="tags">
    {% for tag in node.tags %}
    <a href="{{ BASE_PATH }}{{ site.JB.tags_path }}#{{ tag }}-ref"  class="btn btn-success btn-xs">
      {{ tag }} <span class="badge">{{ site.tags[tag].size }}</span>
    </a>
    {% endfor %}
  </div>
</div>
{% endfor %}

转义大括号：

{% raw %}
你的代码
